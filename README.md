# NN-for-promotors-prediction

Данный репозиторий посвящён проекту по использованию нейронных сетей различных архитектур с целью поиска в последовательностях ДНК регуляторных участков генома - промоторов

В качестве рассматриваемых архитектур были выбраны свёрточные нейронные сети (**CNN**), комбинация свёрточных и реккурентных нейронных сетей (**CNN + RNN**) и предобученные большие языковые модели (**LLM**). В рамках последнего подхода проведено сравнение двух методов дообучения фундаментальной геномной модели **HyenaDNA**: классического обучения классификационной головы (Head Fine-Tuning) и настройки внутренних слоёв с помощью низкоранговых адаптеров (**LoRA**)

## Описание данных

В проекте используется сбалансированная выборка последовательностей ДНК длины 201, разделённая в пропорции 50/50:
*   **Класс 0:** Промоторные последовательности ДНК, извлечённые из референсного генома человека GRCh38
*   **Класс 1:** Случайные непромоторные участки генома аналогичной длины

Все файлы, необходимые для получения данных, расположены по [ссылке](https://drive.google.com/drive/folders/1NmX6wfqxDa-6k8tFaLCZdLVDwInPVU2Y?usp=sharing)

## Описание методов

Для построения модели классификации участков генома были проведены следующие эксперименты:

1.  **CNN**
    *   Последовательности генома были представлены в виде матриц размерности $4 \times 201$ с помощью one-hot кодирования нуклеотидов
    *   Модель с помощью свёрточных и полносвязаного слоёв училась отличать промоторные участки
2.  **CNN + RNN**
    *   К предыдущей архитектуре был добавлен дополнительный слой LSTM для усиления информативности признаков
3.  **HyenaDNA + Fine-Tuning**
    *   Все веса основной части модели полностью замораживаются
    *   Градиентный спуск обновляет только параметры финального линейного слоя классификации
    *   Модель выступает в роли фиксированного экстрактора признаков
4.  **HyenaDNA + LoRA**
    *   Внутрь всех линейных проекций внедряются обучаемые низкоранговые матрицы адаптера LoRA ($r=8, \alpha=16$).
    *   Обучение адаптирует внутренние слои модели под специфику задачи бинарной классификации участков ДНК

Для экспериментов 3 и 4 в качестве базовой архитектуры выбрана фундаментальная геномная модель **HyenaDNA** (`hyenadna-tiny-1k-seqlen-hf`), предобученная на больших массивах ДНК-последовательностей

## Результаты и выводы

Оценка качества моделей проводилась на отложенной тестовой выборке по значениям Loss и Accuracy

### Сводная таблица результатов

| Модель | Train Loss (%) | Test Loss (%) | Train Acc (%) | Test Acc (%) |
| :--- | :---: | :---: | :---: | :---: |
| **CNN** | 28.22 | 50.73 | 88.88 | 87.22 |
| **CNN + RNN** | 29.97 | 34.63 | 88.62 | 87.66 |
| **LLM + FT** | 37.19 | 35.72 | 0.83.57 | 83.95 |
| **LLM + LoRA** | 19.87 | 23.26 | 92.2 | 90.7 |

### Основные выводы:
*   Добавление слоя LSTM к свёрточным слоями помогло снизить loss на отложенной выборке на ~15%
*   Метод с использованием **LoRA** показал наиболее высокую accuracy по сравнению с другими моделями, т. к. адаптеры позволяют модели глубже перестраивать представления ДНК-мотивов во внутренних слоях, не ограничиваясь только выходной головой
*   Этот же метод показал наиболее низкие значения loss как при обучении, так и при валидации
